Ini adalah Proyek Klasifikasi Gambar dengan dataset Lung Cancer (Histopathological Images). Dataset dilatih dengan pembagian data menjadi train, test, dan validation.

Dataset terdiri dari 3 kelas yaitu Adenocarcinoma, Benign, dan Squamous_cell_carcinoma. Masing-masing class terdiri dari 5000 files data.

Pekerjaan dilakukan di google colaboratory oleh Nicholas Rayden

Akurasi yang diperoleh adalah:
Test Accuracy: 0.9771
Training Accuracy: 0.9684

url: https://www.kaggle.com/datasets/rm1000/lung-cancer-histopathological-images
